## Shrink diff

The code diff is 121 insertions and 227 deletions across 19 Go files: **106 net lines removed**,
including 103 production lines and three test lines. Production LOC falls from 9,096 to 9,028 in the
AI mirror and its runtime helpers, and from 9,653 to 9,618 in the coding-agent mirror, CLI, and
truncation helper.

- Four content-slice decoders now share one typed decoder while retaining the same concrete block
  factories, unknown-block round trips, and error contexts.
- Four identical WTF-8 surrogate decoders became one `internal/jsonwire` helper.
- An unused request argument, an unused variadic path option, an identity closure, a constant-only
  truncation parameter, and a one-field tool-selection wrapper were deleted. The direct
  upstream-spelled `SessionManager.GetCwd` API remains for source compatibility.
- Five hand-written equality scans now use the standard library's `slices.Contains`.

Targeted tests passed after each reduction. The complete fixture, race, vet, lint, and cross-build
matrix below passed on the final source tree. No assertion, expected value, or fixture changed;
test-only edits followed reduced signatures or removed repeated helper arguments.

## Dead code and duplication

`.tools/bin/staticcheck -checks=U1000 ./...` and golangci-lint's production `unparam` pass are
clean. `deadcode -test ./...` reports exported Agent hooks and methods, exported session-manager and
provider APIs, sealed-union marker methods, and the `ComputeEditDiff` upstream convenience surface.
Those are externally reachable public contracts or required upstream seams that whole-program
reachability cannot see. In particular, `GetCwd` is retained because deleting the already-landed
upstream-spelled method breaks Go SDK callers; no actionable unexported unreachable function
remains.

`dupl -t 80` fell from five clone groups to three. The duplicate content-container decoders and
WTF-8 helpers are gone. The remaining structural matches are deliberately separate protocol
branches: text and thinking streams emit different event types and partial shapes; fixed-shape
wire decoders retain named fields and exact error paths; and Agent versus faux-provider cloning has
different unknown-content and argument-normalization behavior. A callback or reflection layer for
those branches adds code and indirection without merging a shared helper.
