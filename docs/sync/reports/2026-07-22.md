## Landed behavior

- The reusable retry helper now drives normal assistant calls, compaction, and branch
  summarization. Retries preserve exponential backoff, cancellation, non-retryable errors, attempt
  accounting, and the scheduled/start/finished callback order from `packages/ai/src/utils/retry.ts`.
- Compaction and branch summaries expose `compaction_retry_*` and `branch_summary_retry_*` events
  through session listeners, JSON/RPC serialization, and the interactive status indicator. The Go
  runtime keeps the original compaction core unchanged and wraps its completion call, matching the
  upstream harness boundary.
- Agent-core accepts an explicit stream function or the process default installed by coding-agent,
  restoring pre-0.81 extension compatibility. The sole API-shape adaptation is error timing: the
  TypeScript constructor can throw when neither exists, while Go's fixed `NewAgent` signature
  reports the identical message at the first prompt or low-level loop call.
- Automatic model-catalog refresh is deferred until interactive startup has rendered, remains
  non-blocking, and keeps the existing cache-only and RPC paths.
- Kimi K3 inherited models use OpenAI thinking format and reasoning effort. The generated catalog
  matches the published v0.81.1 package: 37 providers and 1,098 models, including the new Gemini
  entries and exact Google/OpenCode/OpenRouter/Vercel deltas captured in
  `ai/models/testdata/v0.81.1-model-deltas.json`.
- Releases now publish a deterministic checksummed source archive. CI validates archive contents,
  rebuilds the static Go product from that archive with `-buildvcs=false` before publication, and
  verifies that the publish pass regenerated the same bytes. Disabling VCS stamping is required
  because the verified archive deliberately excludes `.git` metadata.
- Version identity, embedded upstream changelog, catalog data, and all conformance manifests were
  regenerated from the exact target. F10 payloads remain byte-identical; only its manifest
  provenance changed. WP450 changes only the upstream banner to v0.81.1, while the F12 changelog
  command reflects the new embedded release notes.
